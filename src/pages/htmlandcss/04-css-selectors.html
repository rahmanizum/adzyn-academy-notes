CSS Selectors

CSS selectors are used to select the HTML elements that we want to style.

Types of Selectors

  • Element Selectors
  • Id Selectors
  • Class Selectors
  • Attribute Selectors
  • Combinators

Element Selectors

Element selectors are used to select HTML elements by their name. For example, p selects all <p> elements.

Example:

            p {
                color: blue;
            }

Id Selectors

Id selectors are used to select HTML elements by their id. For example, #header selects the HTML element with the id header.

Example:

            #header {
                background-color: yellow;
            }

Class Selectors

Class selectors are used to select HTML elements by their class. For example, .container selects all HTML elements with the class container.

Example:

            .container {
                width: 80%;
            }

Attribute Selectors

Attribute selectors are used to select HTML elements by their attributes. For example, a[href] selects all <a> elements that have a href attribute.

Example:

            a[href] {
                text-decoration: none;
            }

Combinators

Combinators are used to combine multiple selectors. For example, div p selects all <p> elements that are descendants of a <div> element.

Example:

            div p {
                font-size: 16px;
            }
