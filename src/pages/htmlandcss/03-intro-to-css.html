Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used for describing the presentation of a document written in HTML or XML. CSS defines how elements should be rendered on screen, paper, or in other media.

Why Use CSS?

  • Separate content from design
  • Easier to maintain and update
  • Improved website performance
  • Better accessibility and user experience

Types of CSS

CSS can be applied in three ways:

  1. Inline CSS: Applied directly within an HTML element using the style attribute.
  2. Internal CSS: Defined within a <style> tag in the <head> section of the HTML document.
  3. External CSS: Defined in a separate file with a .css extension and linked to the HTML document.
